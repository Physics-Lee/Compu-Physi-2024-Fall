# Stochastic Process

* `21:40` Random Process
  * Definition: $X(w,t)$
    * $X(w)$: 概率论
    * $X(t)$: 物理，微分方程
    * $X(w,t)$: 随机过程，随机微分方程
  * Example of 4 types
    * X dis, t dis: Random Walk
    * X dis, t con: Counting Process
    * X con, t dis: Daily Temperature
    * X con, t con: Brownian Motion
